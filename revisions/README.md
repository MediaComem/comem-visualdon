# Révisions | Visualisation de données

Guide de révisions pour l'examen final du cours VisualDon (HEIG-VD / COMEM+).

---

## Examen type

**Durée :** 2 heures

**Documents autorisés :** un formulaire A4 recto verso, manuscrit ou réalisé à l'aide d'un ordinateur.

**Chapitres non évalués :** Scrollytelling (cours 11).

Un [examen type](./examen-type.md) est disponible pour vous entraîner. Le corrigé sera mis à disposition en fin de semaine.
